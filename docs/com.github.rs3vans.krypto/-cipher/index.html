Cipher

class Cipher

A Cipher is used to encrypt and decryptByteArray data using...

  • a Key
  • an algorithm
  • a block cipher mode (see here)
  • a padding mode (see here
  • optionally a security provider

Note: Encryptions are randomized using an initialization vector (IV).
